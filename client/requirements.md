## Packages
framer-motion | Smooth animations and draggable window interactions
react-hook-form | Form state management for the Contact app
@hookform/resolvers | Zod validation integration for forms
date-fns | Formatting taskbar clock

## Notes
- Using framer-motion for the OS window dragging and animations instead of react-rnd for better compatibility.
- Static images for the OS UI (profile pics, boot logos) will use Unsplash placeholder URLs.
- The retro vaporwave aesthetic relies heavily on CSS gradients and background patterns defined in index.css.
